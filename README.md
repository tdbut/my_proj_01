# RealWorld Tests

Автотесты для сайта https://realworld.qa.guru/ на Playwright.


## Тесты

1. **articles are visible** — проверка списка статей на главной
2. **click on tag** — фильтрация статей по тегу
3. **open article** — переход на страницу статьи
4. **open author profile** — переход в профиль автора
5. **pagination works** — переключение на вторую страницу
